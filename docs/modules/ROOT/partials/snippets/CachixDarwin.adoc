Save `binary-caches.json` on the target machine and install it by running:

[source,shell]
----
$ sudo install \
    -o hercules-ci-agent  \
    -m 0600 \
    binary-caches.json \
    /var/lib/hercules-ci-agent/secrets/binary-caches.json
----

And activate via:

[source,shell]
----
$ sudo ./result/bin/darwin-installer
----
